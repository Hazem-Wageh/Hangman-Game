# 🎮 Hangman Game

## 📌 Overview

This project is a fully responsive **Hangman Game** built using **HTML, CSS, and JavaScript**.
It focuses on enhancing front-end development skills through DOM manipulation, game logic, and responsive UI design.

---

## 🚀 Features

* 🎯 Interactive letter guessing system
* ❌ Wrong attempts tracking with hangman drawing
* 🔄 Random word generation
* 📱 Fully responsive design (Mobile / Tablet / Desktop)
* ⚡ Smooth and clean user interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox / Grid)
* JavaScript (ES6)

---
